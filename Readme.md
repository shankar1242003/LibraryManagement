# 📚 Library Management System

A full-stack Library Management System developed using Node.js, Express.js, SQL Server, HTML, Bootstrap, JavaScript, and jQuery.

## 🚀 Features

- Admin Login
- Dashboard
- Books Management (CRUD)
- Students Management (CRUD)
- Issue Books
- Return Books
- Search Functionality
- REST APIs
- SQL Server Database

## 🛠 Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
- Node.js
- Express.js
- SQL Server

## 📂 Project Structure

```
LibraryManagement
│
├── db
├── routes
├── public
├── package.json
└── server.js
```

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Configure the `.env` file.

4. Start the server

```bash
node server.js
```

5. Open your browser

```
http://localhost:3000/login.html
```
