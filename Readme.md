# 📚 Library Management System

A full-stack **Library Management System** built using **Node.js,
Express.js, Microsoft SQL Server, HTML, CSS, Bootstrap, JavaScript,
jQuery, Docker, and AWS EC2**.

## ✨ Features

- Admin Login Authentication
- Dashboard with statistics
- Books Management (CRUD)
- Students Management (CRUD)
- Book Issue & Return
- Search functionality
- RESTful APIs
- Microsoft SQL Server integration
- Dockerized deployment
- AWS EC2 deployment

## 🛠 Tech Stack

**Frontend:** HTML5, CSS3, Bootstrap, JavaScript, jQuery

**Backend:** Node.js, Express.js

**Database:** Microsoft SQL Server

**DevOps:** Docker, Docker Compose, AWS EC2 (Ubuntu)

## 🚀 Run Locally

```bash
git clone https://github.com/shankar1242003/LibraryManagement.git
cd LibraryManagement
npm install
npm start
```

Open: `http://localhost:3000`

## 🐳 Docker

```bash
docker compose up -d
```
